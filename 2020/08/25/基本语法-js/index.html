JS基本语法

1.输入输出

  • 输入： 
    prompt();	//字符串输入  
+prompt();	//数字输入
  • 输出： 
    alert();	//警告框输出
document.write();	//body中输出
console.log();	//控制台输出

2.数据类型与转换

可以通过typeof + 变量名检查变量的数据类型

  • String 字符串
    说明 用单引号或双引号引起来，但不能混用，不能同一嵌套
    特殊符号 " = “
    ' = ‘
    \n = 换行
    \t = 制表符
    \ = \
    强制类型转换 toString()方法：a = a.toString();
    String()函数：a = String(a);

  • Number 数值

    说明 js中所有数值都为Number类型，包括整数和浮点数
    强制类型转换 Number()函数： a = Number(a);
    String()函数：a = String(a);
    parseInt()函数，转为整数：a = parseInt(a);
    parseFloat()函数，转为整数：a = parseFloat(a);
    true = 1, false = 0, Null = 0，undefined = NaN
    其他进制 以0x开头表示16进制数字
    以0开头为8进制数字

  • Boolean 布尔值

    说明 布尔值表示逻辑判断,true 表示真,false 表示假
    Boolean()函数类型转换 Number 除了 0 和 NaN ，其他都为true
    String 除了空串，其他都是true
    Null 和 Undefined 都是false
    Object 转换为true

  • Null 空值

    Null类型的值就只有一个，就是null,表示第一个空对象,typeof检查null时，会返回object

  • Undefined 未定义

    Undefined(未定义) 类型的值只有一个，就是undefind,当声明一个变量，但并不给变量赋值是，它的值就是undefined,typeof检查undefined，会返回undefined

  • Object 对象

3.运算符

  • 一元运算符

    自增：自身基础上增加1 ,自增后，值立即自增1,分为两种：a++ 和 ++a

      var a = 1;
  a++ = 1;
  ++a = 2;    //a++ 和 ++a 的值不同
  a = 2;

    自减同上

  • 逻辑运算符

    • !非：

      一个值进行非运算 对一个布尔值进行取反操作，true –> flase

    • &&与：

      &&与：可以对符号两侧的值进行与运算；
      只有两端的值都为true时，才会返回true。只要有一个false就会返回false；
      与是一个短路的与，如果第一个值是false，则不再检查第二个值；
      对于非布尔值，它会将其转换为布尔值然后做运算，并返回原值；
      规则：
      1.第一个值为false，则返回第一个值；
      2.如果第一个值为true，则返回第二个值。

    • ||或：

      可以对符号两侧的值进行或运算；
      只有两端都是false时，才会返回false。只要有一个true，就会返回true；
      或是一个短路的或，如果第一个值是true，则不再检查第二个值；
      对于非布尔值，它会将其转换为布尔值然后做运算，并返回原值；
      规则：
      1.如果第一个值为true，则返回第一个值；
      2.如果第一个值为false，则返回第二个值。

  • 赋值运算符

    赋值运算符 说明
    = 可以将符号右侧的值赋值给左侧变量
    += a += 5; 相当于 a = a+5; 可连接字符串
    -= a -= 5 相当于 a = a-5
    其他类似 ……

  • 关系运算符

    说明
    符号 > < >= <=
    比较两个值之间的关系，如果关系成立则返回true，关系不成立则返回false。
    如果比较的两个值是非数值，会将其转换为Number然后再比较。
    如果比较的两个值都是字符串，此时会比较字符串的Unicode编码，而不会转换为Number。

  • 相等运算符

    符号 说明
    == 相等，判断左右两个值是否相等，如果相等返回true，如果不等返回false (相等会自动对两个值进行类型转换，如果对不同的类型进行比较，会将其转换为相同的类型然后再比较，转换后相等它也会返回true。)
    != 不等，判断左右两个值是否不等，如果不等则返回true，如果相等则返回false (不等也会做自动的类型转换。)
    === 全等，判断左右两个值是否全等，它和相等类似，只不过它不会进行自动的类型转换 (如果两个值的类型不同，则直接返回false)
    !== 不全等，和不等类似，但是它不会进行自动的类型转换，如果两个值的类型不同，它会直接返回true
    特殊的值 由于undefined衍生自null，所以null == undefined 会返回true。但是 null === undefined 会返回false。
    NaN不与任何值相等，报告它自身 NaN == NaN 返回false
    判断一个值是否是NaN，使用isNaN()函数

  • 三元运算符 ?

    语法：条件表达式?语句1:语句2;
    执行流程：

    先对条件表达式求值判断;
    如果判断结果为true，则执行语句1，并返回执行结果
    如果判断结果为false，则执行语句2，并返回执行结果

  • 运算符优先级

    和数学中一样，JS中的运算符也是具有优先级的，比如 先乘除 后加减 先与 后或;
    具体的优先级可以参考优先级的表格，在表格中越靠上的优先级越高，
    优先级越高的越优先计算，优先级相同的，从左往右计算。
    优先级不需要记忆，如果越到拿不准的，使用()来改变优先级。

4.流程控制语句

程序都是自上向下的顺序执行的，通过流程控制语句可以改变程序执行的顺序，或者反复的执行某一段的程序。

  • if（条件判断语句） 
    if(条件表达式){
      语句...
  	}else if(条件表达式){
      语句...
  	}else if(条件表达式){
      语句...
  	}else{
      语句...
  	}	//自上到下判断
  • switch（条件分支语句） 
    switch(条件表达式){
      case 表达式:
          语句...
          break;
      case 表达式:
          语句...
          break;
      default:
          语句...
          break;
  	}	//依次将 case后的表达式的值 和 switch后的表达式的值 进行 全等比较
  • while 循环 
    while(条件表达式){
	语句...
	}	//先判断再循环
  • do while 循环 
    do{
 语句...
}while(条件表达式)	//先判断再循环，至少循环一次
  • for 循环 
    for(①初始化表达式 ; ②条件表达式 ; ④更新表达式){
 	③语句...
}	//①-->②-->③-->④
  • break 与 continue

    break可以用来退出switch或循环语句
    continue可以跳过当次循环
    不能再if语句中使用
